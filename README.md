# homebrew-tap

A custom Homebrew tap for tools published by Ahmet Bedir.

## Installation

```bash
brew tap ahmetbedir/tap
brew install <formula-name>
```

## Available Formulae

- `<formula-name>` — short description
- `<formula-name-2>` — short description

## Upgrade

```bash
brew update
brew upgrade <formula-name>
```

## Uninstall

```bash
brew uninstall <formula-name>
brew untap ahmetbedir/tap
```

## Verify

```bash
brew tap
brew search ahmetbedir/tap
```
